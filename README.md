# @azesmway/react-native-rutube

React Native Rutube plugin

## Installation

```sh
npm install @azesmway/react-native-rutube
```

## Usage

```js
import { ReactNativeRutubeView } from "@azesmway/react-native-rutube";

// ...

<ReactNativeRutubeView color="tomato" />
```
